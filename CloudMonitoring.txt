Monitoring Summary 

- CloudWatch: 
    - Metrics: monitor the performance of AWS services and billing metrics 
    - Alarms: automate notification, perform EC2 action, notify to SNS based on metric 
    - Logs: collect log files from EC2 instances, servers, lambda functios...
    - Events (or EventBridge): react to events in AWS, or trigger a rule on a schedule 

- CloudTrail: audit API calls made within yout AWS account
- CloudTrail Insights: automadated analysis of yout CloudTrail Events 
- X-Ray: trace requests made through yout distributed applications 
- AWS Health Dashboard: AWS events that impact yout infrastructure 
- Amazon CodeGuru: automated code reviews and application performance recommendations
